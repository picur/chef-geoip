chef-geoip
===========

Installs GeoIP database and lookup executables.

---
Requirements
============

Platform 
---------

* Ubuntu, Debian

Tested on Ubuntu 12.04.

Cookbooks
---------

* apache

---
Attributes
==========

* `node[geoip][]` = Description

Usage

The default recipe will install the geoip lookup binaries and the GeoLite Country databases.
